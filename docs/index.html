Three verbs. Infinite designs.

From raw nodes to full marketing campaigns, all from your terminal.

Generate

36 production-ready templates. Carousels, pitch decks, OG images, storyboards, UI screens — one command each.

figma-kit make carousel -t noir

Compose

28+ mid-level patterns. Cards, buttons, effects, auto-layout — build custom designs by composing primitives.

figma-kit card glass --preset strong

Audit

10 automated QA checks. Contrast ratios, touch targets, orphaned layers, font consistency — catch issues before handoff.

figma-kit qa checklist --page 0

How it works

You run
figma-kit
→
Generates
JavaScript
→
Executed by
use_figma MCP
→
Creates in
Figma

The CLI generates Figma Plugin API JavaScript. The AI feeds it to the official Figma MCP server. No CDP hacking, no browser automation — just the official API.

Built-in themes

Default Dark

Tech/SaaS aesthetic. Blue-teal accents on deep navy.

Light

Print-friendly. Clean whites and subtle greys.

Noir

Dark premium brand theme. Primary blue #3366FF, Inter + Geist Mono.
