How This Site Was Built
This isn’t a mockup. We described what we wanted in natural language — AI used figma-kit commands to build every element in Figma. Then the same design became this marketing page.
1

Asked AI to create a Figma file

Used create_new_file from the Figma MCP server.

2

Prompted “Set up 3 pages”

AI ran page create and page rename for Landing, Components, Design System.

3

Described the hero section

AI composed node create frame, fx mesh, text create, ui button, ui badge.

4

“Add 3 glass feature cards”

AI called card glass -t noir three times. Each got fx glow underneath.

5

“Create 19 UI components”

Buttons, badges, toggles, checkboxes, radios, tabs, dropdowns, tables.

6

“Generate the design system”

One command: ds create -t noir. Color swatches, type scale, spacing bars.

7

“Push tokens to Figma variables”

ds variables-create pushed 11 color tokens into Figma’s variable system.

8

“Export CSS tokens”

export tokens --format css prints clean :root CSS custom properties.

9

“QA everything”

qa checklist walks every node. 258 nodes, 0 issues.

10

Turned the Figma design into this site

Colors, fonts, layout — 1:1 match of this Figma file.
