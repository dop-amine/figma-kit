**First comment (post immediately after submission):**

Hey HN! I built figma-kit because I wanted to describe a website to an AI agent and have a complete Figma design appear.

**What it looks like in practice:**

I paste stripe.com into Cursor and say: "Create a figma-kit theme that matches this site, then build a landing page with a hero, feature cards, and pricing."

The AI extracts the colors (#0A2540, #635BFF, #00D4AA), runs `figma-kit theme init` to create a theme, then sequences `make screen`, `card glass`, `ui button`, and `fx mesh` commands. Sixty seconds later, a complete branded landing page is in my Figma file.

**The tool:** figma-kit is a single Go binary with 120+ commands that generate JavaScript for Figma's official MCP server. It covers everything from `node create frame` (low-level) to `make pitch-deck --slides 7 --content deck.yml` (high-level). A built-in theme system lets you create themes from any reference — paste a URL, screenshot, or color description, and the AI extracts colors into a complete figma-kit theme.

**What makes this different from existing tools:**
- **Prompt-driven**: You describe what you want in natural language. The AI picks the right commands. You never touch the CLI directly unless you want to.
- **Reference-to-design pipeline**: Paste a website URL → AI extracts colors → creates theme → builds design. All in one conversation.
- **120+ commands across 8 layers**: From primitives (node, style, text, layout) to complete deliverables (carousel, pitch deck, dashboard, OG image), plus design systems, QA, and export.
- **Self-documenting binary**: `figma-kit cookbook` shows 15 real prompt sessions. `figma-kit examples --dump` gives you starter YAML files. The binary teaches you how to use it.
- **Official Figma MCP**: Uses Figma's own MCP server, not browser automation or CDP hacking.

**The marketing site was built using figma-kit itself** — every frame, card, text node, and color was generated by AI + figma-kit commands. The HTML is a 1:1 match of the Figma file.

**Architecture:** Pure Go binary, no runtime dependencies. JavaScript is embedded via `go:embed` — it's the output format for Figma's Plugin API, not implementation code. Themes, examples, and the full prompt cookbook are all embedded in the binary.

**Install:** `brew install dop-amine/tap/figma-kit` or `go install github.com/dop-amine/figma-kit@latest`

Works with Cursor, Claude Code, or any MCP-compatible AI agent. Also runs standalone from the terminal.
